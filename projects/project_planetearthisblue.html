Planet Earth Is Blue

GMTK GameJam 2020 (48h) - Theme : Out Of Control

My submission for the GMTK Game Jam 2020. I had 48h to make this game. I went for a chill, gravity based game in space.

Key points

  • Unity
  • Game Jam
  • Made in 48h
Game Jam page Play the game

Theme : Out Of Control

The astronaut is helpless in space. Help him to reach the Earth by creating planets and pulling him with gravity back home. It's a very simple game, with very little emphasis on the objective or the gameplay, but much more on the calming ambiance.

My goal with this game was to spend as much time of the jam on polishing the game to make it as clean as possible. I managed to have the game "finished" halfway through the week end so I could spend the other half on adding animations, sounds and effects. I managed to save a lot fo time by figuring out quickly how to have nice unique planets using a simplex noise in a shader. I initialy went for a realistic gravity, but it was too hard to play with so I ended up doing a simpler version that simply speed up the astronaut and curve its trajectory when nearby a planet.

Results

The game placed 84th out of 5400 entries overall (123rd in graphics, 211th in fun and 297th in originality). It did lack a bit of originality, but that wasn't what I was going for at all with this game so I'm fully satisfied with the result. I believe the very polished aspect of the game did wonders on the ratings
