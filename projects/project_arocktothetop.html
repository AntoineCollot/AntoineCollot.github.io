A Rock to the Top

GMTK2024 (4 Days) - Theme : Built to Scale

My submission for the GMTK 2024 Gamejam. I had 4 days to make this game. I made a challenging, precision driven game focused on local versus.

Key points

  • Unity
  • Game Jam
  • Made in under 48h
  • Action

Game Jam page Play the game

Theme : Built to Scale

This theme could lead to some really cool ideas. I couldn't quite get one doable in the time constraint tho, so I went for a more classic idea. The player has to build tower, always taller. To do this, they must climb at the top to build a new floor, but as the tower gets highter it gets harder and harder.

Climbing

The main mechanic is the climb. It's based on two parts: Jumping and grabbing a hold. Most of the difficulty comes from the jump. The player has to use the correct strength and aim to have the right trajectory toward the next hold. Then, when close to a hold, they must press a button to grab it. My intent was to create a rythme jump-grab-jump-grab while playing.

Hand painting

I recently practiced handpainting, so I wanted to use this graphic style for this game. I'm really happy of how the game looks, but it was fairly time consumming in the end, and not really fit for a jam. Players did not really pay attention to it, so it was probably not worth the time.
