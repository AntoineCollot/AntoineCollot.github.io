Last Breath

Ludum Dare 47 Compo (48h) - Theme : Stuck In A Loop

My submission for the Ludum Dare 47 Compo. I had 48h to make this game. I focused mostly on the 3D for this game, to create a cute environment.

Key points

  • Unity
  • Game Jam
  • Made in 48h
Game Jam page Play the game

Theme : Stuck In A Loop

I thought of a train, since a train always loop. I first decided of a way to control the path of the train, you would click on a tile and change the type of rail (from horizontal to vertical to a turn). I then needed something that would force the player to not let the train looping forever along the same path.

I added bandit cells that the train should avoid, station that the train would have to go to regularly, passengers that the train should pick up and finally randomly changing rails. I should have instead focused on having only one mechanic. Because of that I ran out of time before polishing the game as much as I wanted to.

Results

The game placed 116th overall out of 800 and 7th in graphics. I achieved my goal with the graphics, but even tho the overall result is still pretty good, I find the game to not be that great to play. It's fine for a game jam, but it would need a large rework to be turned into a larger game.
