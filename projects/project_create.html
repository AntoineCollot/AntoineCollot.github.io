Ludum Dare 45 Compo (48h) - Theme : Start With Nothing

My submission for the Ludum Dare 45 Compo. I had 48h to make this game. I took the theme quite literally and made a sandbox game when the player starts with nothing.

Key points

  • Unity
  • Game Jam
  • Made in 48h
Game Jam page Play the game

Theme : Start with nothing

I took the theme literally, in this game you play as god and have to create the world. The games start with an empty screen, clicking anywhere creates the world, then the player can start creating things. Most things are created by dragging an object onto another.

As in most games I make, I focused on making it feels good and be fun to mess around with.

There isn't really a clear objective with this one, although things the player creates are sorted between bad things and good things, so they may want to avoid creating bad things, or maybe they prefer to try and create as many things as possible.

Results

The game scored 261th (2613 entries - 728 compos) overall. It also score much better in individual categories, with 90th in Fun, 54th in Theme and 41st in Humor (those being my main focus with this game).

I feel like the game is great and is exactly what I wanted to do with this jam. With a little bit more time I could have added some sounds effects, which would have greatly improved its overall rating I believe.
