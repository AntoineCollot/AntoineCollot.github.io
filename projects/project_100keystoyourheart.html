From Game Jam to Full Game

I made a small puzzle platformer game in GMTK Game Jam a few years back. I was only a few minutes long game, but I felt like it had some potential.

I decided to not use the older game as a base, and instead restart from scratch. The gameplay would be mostly the same, but I changed the graphic style from a basic pixel art to a hopefully more unique comic look.

Time constraints

I wanted to only spend a few months on this project. It was quite a challenge, as it felt like having so much more time than the GameJam's 48h of the original game, while still being really short for a few hours long game. I didn't want the game to look amateur'ish either, so I had to polish everything. I had to cut some features and levels in the end, but I manage to finish in time with a very professional looking product.
